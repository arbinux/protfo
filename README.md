# 🌐 Flask Portfolio Project

A dynamic portfolio web application built using Flask.

---

## 🚀 Features

* ⚡ Dynamic Flask Web App
* 🎨 Clean UI with Templates
* 🔗 API Integration (`requests`)
* 🖼️ Image Handling (`Pillow`)
* 🔐 Authentication (JWT)

---

## 🛠️ Tech Stack

* 🐍 Python
* 🌐 Flask
* 🎨 HTML, CSS
* 📡 Twilio API *(optional)*

---

## ▶️ Run Locally

```bash
# Install dependencies
pip install -r requirements.txt

# Run the application
python app.py
```
